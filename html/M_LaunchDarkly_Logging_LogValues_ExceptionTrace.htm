Remarks
Calling ToString() on the object returned by this method returns the exception's stacktrace as a string. This string is not constructed unless ToString() is called, so writing exceptions to the log in this way incurs very little overhead if logging is not enabled for the specified log level.
Examples
try { ... } catch (Exception e) { logger.Debug("Caught: {0} {1}", LogValues.ExceptionSummary(e), LogValues.ExceptionTrace(e)); }
